## smarthome-arduino

Projet PlatformIO

Contrôleur entrée/sortie avec implémentations :  

* Porte de garage :  
  * Capteur ouverture  
  * Capteur fermeture  
  * Commande bouton poussoir  
  * Commande moteur  

* Carillon :
  * Commande bouton poussoir  
  * Commande sonnerie  
  * Gestion d'un "mute" activable à distance  
  * Gestion d'un "mute" via timer pour éviter multiples sonneries intempestives
